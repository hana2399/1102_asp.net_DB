樲蒔二手書籍交易平台
=====
1102 網際網路程式設計 期末專題作業<br>
透過連結database呈現<br>
以線上二手書店作為主題<br>
呈現客戶端頁面<br>
有多種上架書籍管理功能，如分類、編輯、上架以及刪除<br>
<br>
<h3>登入畫面</h3>
登入畫面隨意輸入會跳出有帳號密碼各三組的提示<br>
請選擇帳密對應登入(mary/123)/(kate/456)/(shai/789)<br>
<img width="960" alt="01登入畫面" src="https://user-images.githubusercontent.com/105577291/195973932-b801e3bc-3adc-40eb-aa55-73d7ecdbf115.png">
登入後再點擊「商品檢視」，進入書籍總覽<br>
<img width="960" alt="02登入後畫面" src="https://user-images.githubusercontent.com/105577291/195973947-d2999c60-a0ad-483a-a1ce-28550719e2be.png"><br>
<h3>書籍總覽</h3>
書籍資料總覽，同時也列出書籍IBSN碼、作者和出版日期等資料<br>
共兩頁，下方有分頁功能<br>
第一頁<br>
<img width="960" alt="0301書籍總覽" src="https://user-images.githubusercontent.com/105577291/195973950-2a5a8a87-3699-4c43-9f1f-908e235e7351.png">
第二頁<br>
<img width="959" alt="0302書籍總覽" src="https://user-images.githubusercontent.com/105577291/195973938-3b3bbc1d-7072-454e-8548-e841c7785d21.png">
<h3>書籍分類功能</h3>
所有書籍分五種類<br>
分別為「文學小說」、「人文社科」、「藝術設計」、「自然科普」以及「語言學習」<br>
每個項目附有編輯和刪除功能<br>
<img width="960" alt="0401書籍分類" src="https://user-images.githubusercontent.com/105577291/195973939-a4f06b30-f1ea-47b9-bf70-af7e2b24c462.png">
點進編輯之橘黃色按鈕可對書本進行編輯<br>
<img width="960" alt="05書籍編輯" src="https://user-images.githubusercontent.com/105577291/195973948-7d400ae6-a907-494a-8d70-a322ab1c1305.png">
<h3>書籍上架功能</h3>
原本的自然科普頁面<br>

點選導覽列之「上架二手書」之按鈕，引導至上架書籍頁面<br>
輸入好上架書籍之資料，並點選新增按鈕<br>
